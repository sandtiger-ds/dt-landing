Does it connect to my bank?

No. For privacy and simplicity, Dollar Tracker is manual-entry. That means no syncing or data transmission issues and better awareness of your spending.

Is my data private?

Yes. Your data resides in your browser's storage and never leaves your device. Also, your data is directory dependant, so running DT from a different folder will not pull up the data you stored from your original run.

What platforms are supported?

Any OS running on a Linux, Apple, or Windows machine. Since DT is browser-based, it is machine/OS independant.

Is Dollar Tracker reliable?

Yes, as long as you run the html file from the same directory every time, your data will always be there.

Can I use multiple instances of Dollar Tracker?

Yes. Just copy the files to a new directory and all the data will be reset for a new instance.

Can I get a mobile version?

Dollar Tracker shines as a desktop app, but you could run it on a tablet or phone if you copied the files over and ran it from your mobile device's browser.

Is it easy to setup/install?

Actually, Dollar Tracker doesn't need any 'installation' per se, just copy the three web files to a folder and run index.html. Remember to run from the same folder the next time, since the browser will look for your data in that same location.

Is it easy to learn to use?

Dollar Tracker is very intuitive and easy to use, but to make it even easier, we do provide a text document that walks you through general usage in detail, and gets you up to speed fast.
